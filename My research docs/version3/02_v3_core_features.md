# Version 3: Core Features & Enhancements

यह दस्तावेज़ Version 3 में जोड़े गए नए कोर फीचर्स की लिस्ट है।

## 🚀 Priority 1: Advanced Invoicing
**✅ 1. Smart PDF Builder**
- **समस्या:** पुराने इनवॉइस जेनरेटर में लाइव प्रीव्यू और कस्टमाइज़ेशन की कमी थी।
- **समाधान:** एक टू-पेन (Two-pane) लेआउट बनाया गया जहाँ बाईं ओर फॉर्म और दाईं ओर लाइव PDF प्रीव्यू है।

**✅ 2. Multiple Line Items Support**
- **समस्या:** पहले इनवॉइस में केवल एक ही आइटम (title, description, amount) होता था, जिससे मल्टीपल सर्विसेज़ का बिल बनाना मुश्किल था।
- **समाधान:** डेटाबेस में `line_items` (JSONB) कॉलम जोड़ा गया और UI में डायनामिक तौर पर नए आइटम्स जोड़ने, डिलीट करने और टोटल कैलकुलेट करने का फीचर दिया गया।

**✅ 3. Dynamic PDF Calculations & Render**
- **समस्या:** PDF में आइटम्स और उनका कुल योग (Total) डायनामिक तरीके से नहीं दिखता था।
- **समाधान:** `@react-pdf/renderer` का इस्तेमाल करके लाइव टेबल रेंडरिंग और ऑटोमैटिक कैलकुलेशन लागू की गई।

**✅ 4. Supabase Storage Policies & Public URLs**
- **समस्या:** स्टोरेज बकेट्स (Buckets) में इमेजेज और PDF प्राइवेट होने के कारण प्रीव्यू नहीं दिख रहा था।
- **समाधान:** `ai-knowledge-base` (या अन्य) बकेट्स को पब्लिक किया गया और RLS पॉलिसीज अपडेट की गईं।
